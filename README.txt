This project features 2 different predictors on different data sets (canned and uncanned). 

LR_NB_predictor.ipynb contains the predictor on canned data taken from UCI. This predictor uses Logistic Regression and Naive Bayes from scikit-learn.
nb_predictor.ipynb contains the predictor on uncanned data taken from the US Census Bureau. This predictor uses self-implemented Naive Bayes.

To run, open the .ipynb files and follow directions within those files.

Data: 
- the 'data' folder contains csv files for nb_predictor.ipynb
- adult.data.csv is the data file for LR_NB_predictor.ipynb

Other file:
- dictionary.py is a file used by nb_predictor.ipynb that maps features to their abbreviation for quick querying purposes.
